# kyodokoiku
かんたんな質問に答えると、それに応じたおすすめスポットを吐き出してくれるアプリ。

<h2>意識した点</h2>
<li>MVCモデルを意識し、できるだけViewとロジックを分けて設計した。</li>

<h2>機能</h2>
<li>ログイン、ログアウト、ログイン維持機能</li>
<li>MySQLを使ったCRUD機能</li>
<li>認証機能</li>
<li>コンテンツ編集におけるバリデーション機能</li>

<h2>技術</h2>
<li>MySQL</li>
<li>PHP</li>
<li>Bootstrap</li>
<li>Docker</li>
